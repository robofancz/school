CSS

Cascading Style Sheets - Kaskádové styly

Formátovací jazyk pro popis způsobu zobrazení HTML prvků

Hlavním smyslem jazyka je oddělení vzhledu dokumentu od jeho struktury a obsahu

Jazyk CSS není schopen vytvořit objekt, pouze ho formátuje a tím pádem není schopen fungovat samostatné bez jazyka HTML

Jazyk CSS obsahuje pouze "vlastnosti" a "hodnotu" (narozdíl od HTML neobsahuje "příkazy"). Hodnota se vlastnosti přiřadí pomocí dvojtečky a jednotlivé vlastnosti jsou od sebe odděleny středníkem

Existují tří způsoby zápisu kaskádových stylů:

1. Přímý styl

Nejstarší, nejméně praktický a nejméně přehledný způsob zápisu, který se dnes používá jen velmi zřídka

idea oddělení formátu od obsahu zde ještě není zcela dpdržena

formátovací pokyny se aplikují pouze na jeden konkrétní objekt - výrazně pomalejší a méně přehledné než ostatní způsoby

pokyny se zapisují jako hodnota atributu style formátovaného prvku

2. stylopis v hlavičce

Poznámka:
Nekombinujeme více typů stylopisu dohromady!!!
Přímý styl má přednost před stylopisem v hlavičce a ten má přednost před externím stylopisem. Pokud je jeden prvek definován vícekrát platí definice přímého stylu.

Zapisuje se do hlavičky souboru dovnitř párového příkazu style.

Definice platí pro všechny objekty daného typu nacházející se na stránce (... například pro všechny odstavce)

Objekt definujeme tak, že napíšeme jeho název (příkaz jazyka HTML bez špičatých závorek) a jeho vlastnosti uzavřeme do složených závorek(Každou vlastnost píšeme na nový řádek!)

Externí stylopis

Definice vlastností prvků se zapisují do samostatného souboru s příponou .css

hlavní výhodou je, že tento soubor se dá připojit k více než jedné webové stránce a změna v tomto souboru se projeví na všech stránkách, které jsou na něj napojené

Připojení externího stylopisu se provádí v hlavičce souboru pomocí nepárového příkazu link

struktura externího souboru: soubor má stejnou strukturu jako stylopis v hlavičce, jen neobsahuje párový příkaz style

externí stylopis nekombinujeme s ostatními typy stylopisů. Všechny ostatní zápisy mají přednost před externím stylopisem
